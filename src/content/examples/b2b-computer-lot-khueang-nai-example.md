## ตัวอย่างเคสประเมินราคาคอมบริษัท 8 เครื่องที่เขื่องใน

> **⚠️ หมายเหตุ:** หน้านี้เป็น**ตัวอย่างเคสประเมินราคา** ไม่ใช่ผลงานรับซื้อจริง จัดทำขึ้นเพื่อให้บริษัท ร้านค้า และหน่วยงานในย่านเขื่องในเห็นภาพว่าการรับซื้อคอมยกล็อตทำงานอย่างไร และควรเตรียมอะไรก่อนติดต่อ ราคาจริงขึ้นอยู่กับสภาพสินค้าและราคาตลาดช่วงนั้น

การขายคอมพิวเตอร์ยกล็อตต่างจากการขายทีละเครื่องอย่างสิ้นเชิง มีทั้งข้อดีคือประหยัดเวลา และข้อที่ต้องเตรียมให้พร้อมกว่า โดยเฉพาะเรื่องข้อมูลในเครื่องและเรื่องอำนาจในการขายทรัพย์สินองค์กร ตัวอย่างสถานการณ์นี้จะช่วยให้ผู้บริหารหรือฝ่าย ITในอำเภอเขื่องในเข้าใจกระบวนการได้ชัดขึ้น

## รายละเอียดสินค้าในตัวอย่างนี้

ตัวอย่างสถานการณ์นี้เป็นล็อตคอมพิวเตอร์สำนักงานที่มีลักษณะดังนี้:

- **จำนวน:** 8 เครื่อง Desktop PC
- **สเปกคร่าว:** Core i5 Gen 8 / RAM 8GB / HDD 500GB (ส่วนใหญ่) บางเครื่องมี SSD
- **แบรนด์:** คละ (Dell 4 เครื่อง, HP 3 เครื่อง, Acer 1 เครื่อง)
- **จอ:** มี 5 จอที่ใช้งานได้ปกติ 3 จอมีรอยบุบมุมแต่ยังใช้ได้
- **อายุ:** ซื้อมาประมาณ 5-6 ปี ส่วนใหญ่ยังเปิดติดและใช้ได้
- **สภาพ:** ทั่วไปดีแต่มีรอยใช้งาน บางตัวพัดลมเสียงดัง
- **ข้อมูล:** ยังไม่ได้ลบข้อมูลในเครื่อง

## สิ่งที่ร้านจะตรวจเช็กก่อนประเมินคอมยกล็อต

การประเมินล็อตต่างจากเครื่องเดียวที่ต้องดูภาพรวม:

- **สเปกแต่ละเครื่อง** – CPU รุ่นและ Gen, RAM ความจุ, ชนิด Storage (SSD ดีกว่า HDD มากในราคามือสอง)
- **สภาพตัวเครื่องและจอ** – รอย บุบ จอแตก จอแสดงผลปกติหรือไม่
- **เครื่องเปิดติดกี่เครื่อง** – เครื่องเสียเปิดไม่ติดจะตัดออกหรือประเมินราคาต่างกัน
- **อาการที่สังเกตได้** – เสียงพัดลมดัง ร้อนผิดปกติ ช้าผิดปกติ
- **OS และ License** – Windows ที่ติดตั้งมีสิทธิ์ถูกต้องหรือไม่ (มีผลน้อยกว่าสเปก แต่บางคนให้ความสำคัญ)
- **พอร์ตและอุปกรณ์** – USB, HDMI, มีเมาส์คีย์บอร์ดมาด้วยหรือไม่
- **ข้อมูลในเครื่อง** – ยังมีข้อมูลหรือ format แล้ว (ต้องลบก่อนขาย)
- **ราคาตลาดมือสองช่วงนั้น** – Core i5 Gen 8 ราคาผันผวนตามความต้องการในตลาด

## ตัวอย่างวิธีส่งข้อมูลล็อตคอมให้ร้านประเมิน

การส่งข้อมูลล็อตคอมต้องละเอียดกว่าสินค้าชิ้นเดียว:

1. **ถ่ายรูปตัวอย่างเครื่อง** จาก 2-3 เครื่องในล็อต ให้เห็นด้านหน้า หลัง และสภาพโดยรวม
2. **ถ่ายรูป System Properties** ของแต่ละเครื่อง (กด Windows + Pause หรือ Right-click This PC > Properties) เพื่อให้เห็น CPU, RAM, OS
3. **ระบุจำนวนเครื่องทั้งหมด** และแยกว่าเครื่องเปิดติดกี่เครื่อง เครื่องเสียกี่เครื่อง
4. **แจ้งสภาพโดยรวม** อย่างตรงไปตรงมา เช่น พัดลมดัง จอมีปัญหา
5. **แจ้งว่ามีจอและอุปกรณ์เสริมมาด้วยหรือไม่**
6. **แจ้งพื้นที่** และว่าสะดวกให้มาดูที่ไซต์หรือนำมาหน้าร้านในอุบล
7. **รอรับราคาประเมินเบื้องต้น** ก่อนตัดสินใจ

## พื้นที่นัดประเมินและจุดที่เกี่ยวข้องในเขื่องใน

ตัวอย่างสถานการณ์นี้เกิดขึ้นในย่านเขื่องใน ซึ่งอยู่บนเส้นทางอุบล–ยโสธร ห่างจากเมืองอุบลประมาณ 30-35 กม. พื้นที่ที่ลูกค้ามักระบุ เช่น ตัวอำเภอเขื่องใน, ตลาดเขื่องใน, หรือออฟฟิศและร้านค้าในพื้นที่

**สำหรับล็อตขนาดกลาง (8 เครื่อง) สามารถสอบถามจุดนัดที่สะดวกได้ทาง LINE เงื่อนไขขึ้นอยู่กับเวลา ระยะทาง และจำนวนสินค้า** การไปดูที่ไซต์ขึ้นอยู่กับเงื่อนไข ไม่ใช่ทุกกรณีที่เดินทางไปได้

## ปัจจัยที่ทำให้ราคาล็อตคอมเปลี่ยน

ล็อตนี้มีตัวแปรหลายอย่างที่ส่งผลต่อราคารวม:

**ส่งผลดีต่อราคาล็อต:** เครื่องสเปกสม่ำเสมอ, มี SSD แม้บางเครื่อง, CPU Gen 8 ขึ้นไปยังใช้ Windows 11 ได้ (บางรุ่น), เปิดติดทุกเครื่อง, จอดีทั้งล็อต

**ส่งผลลบต่อราคาล็อต:** สเปกคละหลายแบบประเมินยาก, HDD ทั้งล็อต, เครื่องเสียบางตัว, พัดลมดังแสดงว่าใกล้เสีย, CPU Gen เก่ากว่า Gen 6 ราคาตก

## สิ่งที่องค์กรต้องทำก่อนขายคอมยกล็อต – สำคัญมาก

นี่คือส่วนที่หลายองค์กรมักลืมหรือมองข้าม:

**1. สำรองและลบข้อมูลก่อนขายเสมอ**
ข้อมูลในเครื่องทุกเครื่องต้องลบออกหรือ Format ก่อนส่งมอบ เพราะข้อมูลธุรกิจ ข้อมูลลูกค้า และเอกสารสำคัญที่ยังค้างอยู่อาจสร้างความเสี่ยงให้กับองค์กรได้ หลังขายแล้วไม่มีทางเรียกข้อมูลคืน

**2. ตรวจสอบอำนาจในการขายทรัพย์สินองค์กร**
คอมพิวเตอร์ที่เป็นทรัพย์สินของบริษัทหรือหน่วยงาน การจำหน่ายต้องได้รับอนุมัติจากผู้มีอำนาจตามระเบียบขององค์กร ไม่ควรขายโดยที่ยังไม่ได้รับการอนุมัติ

**3. เอกสารที่มีประโยชน์** (ถ้ามี)
ใบกำกับภาษีหรือเอกสารซื้อเดิม บัตรประกัน Serial Number ของแต่ละเครื่อง

## ถ้าเป็นเคสจริง ควรเก็บหลักฐานอะไรบ้าง

- รูปตัวอย่างเครื่องและ System Properties ของแต่ละเครื่อง
- รายการ Serial Number ถ้าต้องการบันทึก
- วันที่รับซื้อและพื้นที่โดยประมาณ
- จำนวนเครื่องและสเปกสรุป
- **ห้ามเปิดเผยชื่อองค์กรหรือข้อมูลที่ระบุตัวลูกค้าได้โดยไม่ได้รับอนุญาต**

## คำถามที่พบบ่อย

**ถาม: Core i5 Gen 8 ราคามือสองตอนนี้ยังดีอยู่ไหม?**
ตอบ: ราคาผันผวนตามตลาด Gen 8 ยังรองรับ Windows 11 ได้บางรุ่น ทำให้ยังมีความต้องการอยู่บ้าง แต่ต้องประเมินจากสเปกและสภาพจริง ส่งข้อมูลมาถามก่อนได้

**ถาม: เครื่อง 8 เครื่องต้องลบข้อมูลทีละเครื่องเองไหม?**
ตอบ: ควรลบทุกเครื่องก่อนขาย ถ้าทำเองไม่ไหว อาจจ้างฝ่าย IT ภายนอกช่วยได้ แต่ไม่ควรส่งมอบเครื่องที่ยังมีข้อมูลอยู่

**ถาม: ร้านจะมาดูที่เขื่องในได้ไหม?**
ตอบ: สอบถามเงื่อนไขทาง LINE ได้เลย การไปดูที่ไซต์ขึ้นอยู่กับเงื่อนไขและจำนวนสินค้า ไม่ใช่ทุกกรณีที่เดินทางไปได้

**ถาม: ถ้าเครื่องบางตัวเสียแล้ว ร้านจะรับทั้งล็อตหรือรับเฉพาะที่ดี?**
ตอบ: ขึ้นอยู่กับเงื่อนไขและสัดส่วนที่เสีย อาจรับทั้งล็อตโดยปรับราคาสำหรับเครื่องเสีย หรือรับเฉพาะที่ดีก็ได้ สอบถาม LINE เป็นกรณีไป

**ถาม: จอมอนิเตอร์รวมในราคาล็อตหรือแยก?**
ตอบ: ประเมินแยกกัน ตัว PC body และจอ แจ้งให้ทราบว่ามีจอกี่จอ สภาพอย่างไร จะประเมินรวมราคาให้ทีเดียว

## สรุป

ตัวอย่างสถานการณ์นี้แสดงให้เห็นว่าการขายคอมยกล็อตต้องเตรียมข้อมูลและดำเนินการหลายอย่างล่วงหน้า โดยเฉพาะเรื่องการลบข้อมูลและการขออนุมัติจากองค์กร ถ้ามีคอมพิวเตอร์สำนักงานต้องการขายในย่านเขื่องใน ส่งรูป System Properties จำนวนเครื่อง สภาพโดยรวม และพื้นที่สะดวก มาทาง LINE เพื่อขอราคาประเมินเบื้องต้นได้ก่อนเสมอ
