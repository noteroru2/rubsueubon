## รับซื้อ MacBook Pro M1 มือสองในอุบลราชธานี

MacBook Pro ที่ใช้ชิป M1 Pro และ M1 Max ซึ่งเปิดตัวปลายปี 2021 เป็นก้าวกระโดดครั้งสำคัญของ Apple ด้วยหน้าจอ Liquid Retina XDR ขนาด 14 และ 16 นิ้ว พอร์ต Thunderbolt 4 ที่กลับมาครบ HDMI SD Card และ MagSafe 3 ที่กลับมาอีกครั้ง MacBook Pro M1 Series ยังคงเป็นที่ต้องการสูงในตลาดมือสองสำหรับผู้ใช้งานหนักและสายสร้างสรรค์

หากคุณมี MacBook Pro M1 ที่ต้องการขายในอุบลราชธานี เราประเมินราคาเบื้องต้นผ่าน LINE ได้ทันทีโดยไม่มีค่าใช้จ่าย ราคาขึ้นอยู่กับรุ่นชิป ขนาดจอ RAM SSD Cycle Count และสภาพเครื่อง

เรามีหน้าร้านจริงในอุบลราชธานี ประเมินอย่างตรงไปตรงมา ชำระเงินหลังตรวจสอบและตกลงราคาเรียบร้อยแล้ว

## รุ่น MacBook Pro M1 ที่รับพิจารณา

MacBook Pro รุ่น M1 มี 3 กลุ่มหลักที่มีราคาตลาดมือสองแตกต่างกัน

**MacBook Pro 14 นิ้ว M1 Pro / M1 Max (2021)**

รุ่น 14 นิ้วที่มาพร้อม MiniLED Liquid Retina XDR Display, Notch, พอร์ตครบทั้ง Thunderbolt 4, HDMI 2.0, SD Card และ MagSafe 3 เป็นรุ่นที่ตลาดมือสองต้องการสูง เพราะพกพาสะดวกกว่า 16 นิ้วแต่สเปคไม่ต่างกันมาก

**MacBook Pro 16 นิ้ว M1 Pro / M1 Max (2021)**

รุ่น 16 นิ้วที่ได้ราคาสูงกว่า 14 นิ้วเนื่องจากหน้าจอใหญ่กว่าและเสียง Speaker ที่ดีกว่า เหมาะสำหรับผู้ใช้งานที่ไม่ได้เน้นพกพา MacBook Pro 16 นิ้ว M1 Max RAM 64GB เป็นรุ่นที่หายากและได้ราคาดีในตลาด

**MacBook Pro 13 นิ้ว M1 (2020) พร้อม Touch Bar**

รุ่นแรกที่ใช้ชิป M1 มี Touch Bar ที่ไม่มีใน 14 และ 16 นิ้ว SSD 256GB ถึง 2TB RAM 8GB หรือ 16GB ได้ราคาต่ำกว่า MacBook Pro 14 และ 16 นิ้ว M1 Pro/Max อย่างชัดเจน แต่ยังได้ราคาดีกว่า MacBook Air M1 เนื่องจากมี Active Cooling และ Touch Bar

## สเปคที่มีผลต่อราคา MacBook Pro M1

**ชิป M1 Pro vs M1 Max**

M1 Pro มี GPU 14 Core หรือ 16 Core และ M1 Max มี GPU 24 Core หรือ 32 Core M1 Max ได้ราคาสูงกว่า M1 Pro พอสมควรเพราะ GPU และ Memory Bandwidth สูงกว่ามาก

**RAM แบบ Unified Memory**

MacBook Pro M1 Pro รองรับ RAM 16GB และ 32GB ส่วน M1 Max รองรับ 32GB และ 64GB RAM สูงกว่าได้ราคาดีกว่า โดยเฉพาะ 64GB ที่หายากและได้ราคาดีมาก

**SSD**

เริ่มตั้งแต่ 512GB ถึง 8TB SSD ขนาดใหญ่กว่าได้ราคาสูงกว่าตามลำดับ SSD 1TB ขึ้นไปเป็นที่ต้องการในตลาดมือสองมากกว่า 512GB

## ปัจจัยที่มีผลต่อราคา MacBook Pro M1 มือสอง

**Cycle Count** Apple ระบุอายุแบต MacBook Pro M1 Series ที่ 1,000 Cycle ตรวจสอบได้ที่ About This Mac > System Report > Power Cycle Count ต่ำกว่า 300 ดีมาก 300 ถึง 600 ยังดี เกิน 700 เริ่มกระทบราคา

**สภาพจอ MiniLED** จอ Liquid Retina XDR ที่ไม่มีรอยขีดข่วน Dead Pixel หรือ Burn-in ใดๆ จะไม่ถูกลดราคา รอยขีดข่วนหรือจุดด่างที่เห็นชัดมีผลต่อราคาพอสมควร

**สภาพบอดี้และพอร์ต** Thunderbolt 4, HDMI, SD Card, MagSafe 3 ต้องทำงานได้ปกติทุกพอร์ต หากมีพอร์ตใดผิดปกติควรแจ้งให้ทราบก่อน

**Touch Bar (เฉพาะ 13 นิ้ว)** Touch Bar ที่ทำงานได้ปกติและ Touch ID ที่แม่นยำ จะไม่ถูกลดราคาในส่วนนี้

**Keyboard และ Trackpad** Magic Keyboard ที่ทุกปุ่มทำงานได้ปกติและ Force Touch Trackpad ที่ไม่มีรอยขีดข่วนชัดเจน จะไม่ถูกลดราคา

## สเปคพิเศษของ MacBook Pro M1 ที่มีผลต่อราคามือสอง

MacBook Pro 14 และ 16 นิ้ว M1 Series มีฟีเจอร์หลายอย่างที่ทำให้แตกต่างจาก MacBook Air M1 อย่างชัดเจน และส่งผลต่อราคาตลาดมือสองโดยตรง

**Liquid Retina XDR Display พร้อม ProMotion 120Hz** จอ MiniLED ที่มี Local Dimming ทำให้ความเปรียบต่างสูงมาก รวมถึง ProMotion ที่ปรับ Refresh Rate ได้ตั้งแต่ 24Hz ถึง 120Hz จอนี้เป็นหนึ่งในเหตุผลหลักที่คนเลือก MacBook Pro แทน MacBook Air

**Active Cooling** MacBook Pro M1 มีพัดลมระบายความร้อน ทำให้สามารถ Sustain Performance สูงๆ ได้นานกว่า MacBook Air M1 ที่ไม่มีพัดลม ฟีเจอร์นี้สำคัญสำหรับงาน Video Render และ Compile Code

**Thunderbolt 4 และ HDMI กลับมา** MacBook Pro M1 14 และ 16 นิ้วมีพอร์ต Thunderbolt 4 สามพอร์ต, HDMI 2.0, SD Card Reader และ MagSafe 3 ทำให้ยืดหยุ่นกว่า MacBook Air M1 ที่มีเพียง Thunderbolt สองพอร์ต

**ลำโพง Six-Speaker ที่เสียงดีกว่า** ระบบเสียงของ MacBook Pro M1 ดีกว่า MacBook Air M1 อย่างชัดเจน โดยเฉพาะ 16 นิ้วที่มี Six-Speaker System

## เปรียบเทียบ MacBook Pro M1 Pro กับ M1 Max ในตลาดมือสอง

ผู้ที่กำลังขาย MacBook Pro M1 ควรรู้ว่าชิปมีผลต่อราคามากแค่ไหน

**M1 Pro 10-Core CPU / 16-Core GPU** เป็นตัวเลือกพื้นฐานของ MacBook Pro 14 และ 16 นิ้ว ราคาตลาดมือสองอยู่ในระดับที่ดีและมีจำนวนในตลาดมากที่สุด ทำให้ขายได้ง่ายกว่า M1 Max

**M1 Max 10-Core CPU / 24-32 Core GPU** มี GPU Core เป็น 2 เท่าของ M1 Pro และ Memory Bandwidth สูงกว่ามาก เหมาะสำหรับงาน Video Production 8K และ 3D Rendering ราคาตลาดมือสองสูงกว่า M1 Pro พอสมควร แต่จำนวนในตลาดน้อยกว่าทำให้ขายช้ากว่าเล็กน้อย

**RAM ที่รองรับ** M1 Pro รองรับ 16GB และ 32GB ส่วน M1 Max รองรับ 32GB และ 64GB รุ่น 64GB หายากที่สุดในตลาดและได้ราคาดีเป็นพิเศษ
