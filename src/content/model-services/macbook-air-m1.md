ส่งรูปถ่ายรอบบอดี้เครื่อง ข้อมูล Cycle Count และหน้าจอแสดงสเปค RAM/SSD มาประเมินแนวราคากับเราทาง LINE @buyhub เพื่อทราบราคาเบื้องต้นและข้อเสนอที่เป็นธรรมก่อนตัดสินใจ

## รับซื้อรุ่น/สเปกแบบไหนบ้าง

เรารับซื้อ MacBook Air M1 (จอภาพ Retina ขนาด 13.3 นิ้ว ปี 2020) สเปกต่างๆ:
- สเปกมาตรฐาน: RAM 8GB / SSD 256GB (รุ่นยอดนิยม)
- สเปกอัปเกรด: RAM 16GB / SSD 512GB หรือ 1TB (Customized Spec)
- ทุกสีตัวเครื่อง (Space Gray, Silver, Gold)

## ปัจจัยที่ใช้ประเมินราคา

- **ความจุ RAM และ SSD:** รุ่นอัปเกรด RAM 16GB หรือหน่วยความจำขนาดใหญ่ จะได้รับการพิจารณาเพิ่มมูลค่าประเมินตามสเปก
- **สุขภาพและรอบชาร์จแบตเตอรี่:** Cycle Count ต่ำ สุขภาพแบตสูง จะช่วยรักษามูลค่าราคาประเมินไว้ได้ดี
- **ความสมบูรณ์ของคีย์บอร์ดและหน้าจอ:** ปุ่มกดสัญชาติไทย-อังกฤษพิมพ์ได้สมบูรณ์ หน้าจอ Retina ไม่มีอาการจอดวง แสงขาว หรือคราบจอลอก
- **กล่องและ Adapter ดั้งเดิม:** กล่องแท้ตรงรุ่น อะแดปเตอร์ชาร์จแท้ 30W USB-C และสายชาร์จตรงรุ่นครบถ้วน

## สภาพที่ควรถ่ายรูปส่งให้ร้านดู

- ถ่ายหน้าจอเกี่ยวกับเครื่องในเมนูระบบ Apple **เกี่ยวกับ Mac เครื่องนี้ (About This Mac)**
- ถ่ายสเปคแบตเตอรี่ในเมนูรายงานระบบ **พลังงาน (Power > Cycle Count & Condition)**
- รูปถ่ายแป้นคีย์บอร์ด หน้าจอแสดงผล และฝาหลังปิดตัวเครื่อง
- รูปถ่ายรอยบุบหรือรอยขีดข่วนเด่นชัดที่ขอบบอดี้อะลูมิเนียม (ถ้ามี)

## อุปกรณ์ที่ควรเตรียม

- บัตรประชาชนผู้ขายตามมาตรการกฎหมายธุรกรรม
- ตัวเครื่อง MacBook Air M1
- อะแดปเตอร์ USB-C 30W ดั้งเดิมและสายชาร์จตรงรุ่น
- กล่องและคู่มือที่ตรงกับเลข Serial ของตัวเครื่อง (ถ้ามี)

## อาการที่มีผลต่อราคา

- ระบบสแกนลายนิ้วมือ Touch ID บริเวณมุมขวาบนคีย์บอร์ดเสียหรือใช้ไม่ได้
- แป้นพิมพ์คีย์บอร์ดบางปุ่มค้าง กดไม่ติด หรือหน้าจอลอกสารเคลือบสะท้อนแสง (Staingate)
- พอร์ต Thunderbolt / USB 4 ชำรุดหรือหลวมจนเสียบสายใช้งานยาก
- สภาพจอด่างหรือจอมืดครึ่งซีกเนื่องจากไฟแบ็คไลท์มีปัญหา
