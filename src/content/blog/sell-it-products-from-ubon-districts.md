## รูปพื้นฐานที่ควรส่งสำหรับสินค้าทุกประเภท

เริ่มจากถ่ายรูปตัวสินค้าทุกมุม ภาพรวมด้านหน้า ด้านหลัง ด้านข้าง และจุดที่มีรอยตำหนิ หากมี serial หรือรหัสรุ่นให้ถ่ายเท่าที่สะดวก ไม่จำเป็นต้องส่งข้อมูลส่วนตัวเกินความจำเป็น แต่ควรให้เห็นรุ่นและสภาพพอสำหรับการพิจารณา

ต่อมาควรส่งรูปอุปกรณ์ที่มี เช่น กล่อง สายชาร์จ adapter Apple Pencil Magic Keyboard จอยเกม เลนส์ แบตเตอรี่ แท่นชาร์จ หรือใบเสร็จ หากมีประกันเหลือให้ถ่ายหลักฐานประกอบด้วย ข้อมูลเหล่านี้ช่วยให้ประเมินชุดสินค้าได้ชัดกว่าการบอกเพียงชื่อรุ่น

## iPhone และมือถือ ควรส่งข้อมูลอะไร

สำหรับ iPhone ให้ส่งรุ่น ความจุ Battery Health สภาพจอ ฝาหลัง กล้อง Face ID และสถานะบัญชี Apple ID หากเป็น Samsung หรือ Android ให้ถ่ายหน้า About Phone เพื่อให้เห็นรุ่น RAM ความจุ และแจ้งว่าสแกนนิ้ว กล้อง ลำโพง ไมค์ และพอร์ตชาร์จทำงานหรือไม่

เครื่องที่จอแตก หลังแตก แบตเสื่อม หรือเปิดไม่ติด ยังส่งรูปให้รับพิจารณาตามสภาพจริงได้ แต่ควรแจ้งอาการละเอียด เพราะการประเมินจากต่างอำเภอจะพึ่งข้อมูลที่ลูกค้าส่งมาเป็นหลักก่อนตรวจตัวเครื่อง

## iPad และ MacBook ควรเตรียมอะไร

iPad ควรแจ้งรุ่น ความจุ Wi-Fi หรือ Cellular สภาพจอ การใช้งาน Apple Pencil หรือ Magic Keyboard และสถานะ Apple ID ส่วน MacBook ควรส่งรูป About This Mac, RAM, SSD, ชิป, Cycle Count, สภาพจอ คีย์บอร์ด trackpad และ adapter

ก่อนขายอุปกรณ์ Apple ควรสำรองไฟล์ ออกจากบัญชี ปิด Find My และล้างข้อมูลส่วนตัวก่อนส่งมอบ หากยังไม่แน่ใจ ให้สอบถามขั้นตอนก่อนนัดตรวจ อย่าเพิ่งรีเซ็ตหากยังไม่ได้สำรองไฟล์สำคัญ

## โน้ตบุ๊ก คอมพิวเตอร์ และอะไหล่คอมควรส่งอะไร

โน้ตบุ๊กควรส่ง CPU, RAM, SSD/HDD, GPU, สภาพจอ คีย์บอร์ด บานพับ แบตเตอรี่ และ adapter หากเปิดเครื่องได้ให้ถ่ายหน้า About This PC หรือ Device Manager หากเปิดไม่ติดให้แจ้งอาการ เช่น ไฟเข้าไหม พัดลมหมุนไหม ขึ้นภาพไหม หรือชาร์จเข้าไหม

คอมตั้งโต๊ะหรือคอมประกอบควรถ่ายภายในเคสให้เห็น CPU Mainboard RAM SSD/HDD GPU PSU และระบบระบายความร้อน ถ้าเป็นคอมเกมมิ่งให้แจ้งอุณหภูมิหรืออาการดับเองขณะใช้งานถ้ามี เพราะข้อมูลเหล่านี้มีผลต่อการตรวจรับ

## กล้อง เลนส์ และอุปกรณ์ถ่ายภาพควรส่งอะไร

กล้องควรส่งรูปบอดี้ทุกมุม รุ่นเต็ม จำนวนชัตเตอร์ถ้าทราบ สภาพจอ ช่องมองภาพ hot shoe เมาท์เลนส์ และเซนเซอร์ สำหรับเลนส์ควรถ่ายหน้าเลนส์ ท้ายเลนส์ เมาท์ และชื่อรุ่นบนกระบอกเลนส์ จุดสำคัญคือเชื้อรา ฝ้า ฝุ่น รอยหน้าเลนส์ ระบบโฟกัส และระบบกันสั่น

ถ้าขายเป็นชุด ให้ถ่ายรวมแบตเตอรี่ แท่นชาร์จ สายคล้อง กระเป๋า ฟิลเตอร์ กล่อง และใบเสร็จ เพื่อให้เห็นความครบของอุปกรณ์ก่อนนัดหมาย
