## เช็กสเปกคอมก่อนขาย ทำไมต้องเตรียมข้อมูลให้ครบ

ก่อนขายคอมมือสอง สิ่งที่ทำให้ประเมินราคาเบื้องต้นได้ใกล้เคียงที่สุดคือข้อมูลสเปกที่ชัดเจน คอมหนึ่งเครื่องประกอบด้วยหลายชิ้นส่วน เช่น CPU, Mainboard, RAM, SSD/HDD, GPU, PSU, Case และระบบระบายความร้อน หากแจ้งเพียงว่าเป็นคอมเล่นเกมหรือคอมทำงาน ร้านจะยังประเมินได้กว้างเกินไป

บทความนี้เป็นคู่มือเตรียมข้อมูล ไม่ใช่หน้าบริการหลัก หากต้องการส่งประเมินโดยตรงสามารถดูหน้า [รับซื้อคอมพิวเตอร์ อุบล](/บริการ/รับซื้อคอมพิวเตอร์-อุบล/) หรือ [รับซื้อคอมประกอบ อุบล](/บริการ/รับซื้อคอมประกอบ-อุบล/) ได้

## 1. ดู CPU ให้รู้รุ่นเต็ม

CPU ต้องดูรหัสเต็ม เช่น Intel Core i5-12400F, i7-12700, i9-13900K หรือ AMD Ryzen 5 5600, Ryzen 7 5800X, Ryzen 7 7800X3D เพราะชื่อ i5 หรือ Ryzen 5 เพียงอย่างเดียวไม่พอ รุ่นต่าง generation มีราคาและความต้องการต่างกันมาก สามารถดูรายละเอียดเพิ่มที่ [รับซื้อ CPU อุบล](/บริการ/รับซื้อ-cpu-อุบล/)

## 2. ดู RAM ว่า DDR4 หรือ DDR5

RAM ต้องดูชนิด DDR4/DDR5 ความจุรวม จำนวนแผง และ Bus speed เช่น 16GB DDR4 3200MHz หรือ 32GB DDR5 5600MHz หากเป็นคอมประกอบ ควรถ่ายรูปสติ๊กเกอร์บนแรมหรือหน้าโปรแกรม CPU-Z ถ้าต้องการขาย RAM พร้อม SSD ดู [รับซื้อ RAM SSD อุบล](/บริการ/รับซื้อ-ram-ssd-อุบล/)

## 3. ดู SSD/HDD และสุขภาพไดรฟ์

SSD ต้องแยก SATA, M.2 SATA หรือ NVMe พร้อมความจุ สิ่งที่ควรถ่ายเพิ่มคือ Health, Power On Hours และ Total Host Writes จากโปรแกรมอย่าง CrystalDiskInfo หรือโปรแกรมของผู้ผลิต ก่อนขายคอมควรสำรองและล้างข้อมูลส่วนตัวใน SSD/HDD เสมอ

## 4. ดู GPU ให้รู้รุ่นและ VRAM

GPU มีผลต่อราคาคอมประกอบและ PC Gaming มาก ต้องดูรุ่นเต็ม เช่น GTX 1660 Super, RTX 3060 12GB, RTX 4060, RTX 4070 หรือ Radeon RX series รวมถึงแบรนด์และรุ่นย่อย ถ้าเป็น RTX ดู [รับซื้อการ์ดจอ RTX อุบล](/บริการ/รับซื้อการ์ดจอ-rtx-อุบล/) หากเคยขุดเหมืองให้อ่าน [การ์ดจอขุดเหมืองขายได้ไหม](/บทความ/gpu-mining-before-selling/)

## 5. อย่าลืม Mainboard, PSU, Case และระบบระบายความร้อน

เมนบอร์ดต้องดูรุ่นและ chipset เช่น B450, B550, B660, B760, Z690 หรือ X670 PSU ควรดูแบรนด์ วัตต์ และมาตรฐาน 80 Plus ส่วน Case และระบบระบายความร้อนช่วยให้เห็นภาพรวมการประกอบ รูปที่ควรถ่ายคือด้านในเคส ด้านหลังเคส และด้านหน้าหรือด้านข้างให้เห็นสภาพจริง

## เช็กลิสต์ก่อนส่งประเมิน

- รูปตัวเครื่องด้านหน้า ด้านหลัง และด้านในเคส
- รุ่น CPU, Mainboard, RAM, SSD/HDD, GPU และ PSU
- รูป Task Manager, Device Manager, CPU-Z หรือ Speccy ถ้าเปิดเครื่องได้
- อาการผิดปกติ เช่น เปิดไม่ติด ดับเอง ภาพแตก หรือเสียงดัง
- กล่อง ใบเสร็จ และประกันของอะไหล่ถ้ามี
- แจ้งว่าต้องการขายยกเครื่องหรือแยกชิ้น

## คำถามที่พบบ่อย

### ไม่รู้สเปกคอมต้องทำอย่างไร

ถ้าเปิดเครื่องได้ ให้ถ่ายหน้า Task Manager, Device Manager หรือ System Information ส่งให้ร้านประเมินเบื้องต้น

### คอมเปิดไม่ติดยังเช็กสเปกได้ไหม

อาจดูจากรูปอะไหล่ภายใน กล่อง ใบเสร็จ หรือสติ๊กเกอร์รุ่นบนชิ้นส่วนได้ แต่ราคาต้องตรวจจริงอีกครั้ง

### ควรล้างข้อมูลก่อนขายคอมไหม

ควรสำรองและล้างข้อมูลส่วนตัวใน SSD/HDD ก่อนขายทุกครั้ง

### ขายแยกชิ้นหรือยกเครื่องแบบไหนเหมาะกว่า

ขึ้นกับสเปกและเป้าหมายของผู้ขาย ถ้าต้องการจบง่ายมักส่งประเมินยกเครื่องก่อน แล้วค่อยคุยแยกชิ้นถ้าจำเป็น
